# Death X

Death X places a small red X at the exact position where the player dies.

Markers stay in the level until the configured limit removes the oldest one or the player clears them from the pause menu. Nearby deaths can optionally merge into an existing marker, increasing that marker's color intensity and briefly glowing the affected X.

The normal marker limit slider goes up to 250. For very large levels or testing, the manual override setting accepts values up to 10000.
