# v1.0.4

- Adds a custom Geode icon.
- Improves death marker placement across player gamemodes.
- Allows a manual marker limit override up to 10000 while keeping the normal slider capped at 250.
- Marks the packaged release as macOS-only until other platform binaries are built.

# v1.0.3

- Changes the default start-of-attempt ignore time to 0.5 seconds.
- Replaces pause-menu circle buttons with one small clear icon.
- Changes merge feedback to a same-size glow/fade instead of a scaling pulse.
- Updates mod id to an index-style namespaced id.

# v1.0.2

- Adds configurable deaths-per-color-step setting.
- Adds configurable start-of-attempt ignore time to prevent startup markers.
- Makes merged markers pulse briefly when they absorb a death.
- Shrinks pause-menu icon buttons.

# v1.0.1

- Replaces object-based merging with optional nearby-death area merging.
- Adds configurable merge radius.
- Expands marker intensity colors up to 100 deaths.
- Replaces pause-menu text buttons with compact icon buttons.
- Adds stricter gameplay checks to avoid start-of-level marker spam.

# v1.0.0

- Initial release.
- Adds death-position X markers.
- Adds configurable marker count, size, opacity, and repeated death handling.
- Adds a pause-menu clear button.
